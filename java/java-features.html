Java Features:

Most striking features of java that differs it from any other programming languages are:

  • Simple
  • Portable
  • Robust and Secure
  • Object Oriented
  • Multi-threaded
  • Platform Independent
  • Compiled and Interpreted
  • High Performance
  • Distributed
  • Dynamic

Simple:

Java is relatively easier to understand and learn if you have some previous programming experiences because it has the same syntactical features of C and C++ which makes it easier to understand the language.


Portable:

Portability is simply means that how easily a java program can run on all the other platforms.This feature makes it easier to run the code anywhere once it is written.


Robust and Secure:

Java is very much Robust and Secure because of its strict compile and run time data type checking.

Java has garbage collection which makes efficient use of the memory and exception handling eliminates any risk of crashing the system.


Object Oriented:

Java is a true object oriented programming language that means that everything in java is treated as one Object.


Multi-Threaded:

Java handles multiple tasks simultaneously by using the concept of Threading.


Platform Independent:

Java is a platform independent language because once written and compiled it generates intermediate codes which can further be executed on any other platforms.


Compiled and Interpreted:

Java is both compiled and interpreted means that it holds both the features while executing the code. First the code is compiled to intermediate bytecode and then the java interpreter interpretes the bytecode and generates the machine code.


High Performance:

Java incorporates the intermediate bytecode and uses garbage collection and the concept of threading reduces the overheads and enhances the overall performances.


Distributed:

Java can be used to create applications for the distributed environment of the Internet.


Dynamic:

Java is a Dynamic programming language means at runtime code can be Dynamically linked.
